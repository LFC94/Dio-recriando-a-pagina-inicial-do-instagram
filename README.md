## Recriando a página inicial do Instagram 
Nesse projeto o desafio de reconstruir a página inicial de login do Instagram, no qual será abordado o conceito sobre CSS utilizando Flexbox, uma metodologia de posicionamento de elementos em tela mais utilizada no mercado assim como conceitos de responsividade, além disso a expert disponibiliza todo o material necessário em seu GitHub para que você possa realizar o seu projeto.

Projeto desenvolvido para o bootcamp: [**HTML Web Developer**](https://web.digitalinnovation.one/track/html-web-developer?tab=path)
